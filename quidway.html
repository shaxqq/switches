Quidway

display interface Ethernet 0/0/№ - состояние порта.

display mac-address dynamic Ethernet 0/0/№

На порту пишем virtual-cable-test (после ввода команды нужно подтвердить)

display dhcp snooping user-bind all – посмотреть кто получает по DHCP

display cpu – проверить загрузку процессора на коммутаторе

quit – выйти со свича


Если на свитче есть правила привязок
system-view - заходим в режим конфигурации
interface Ethernet0/0/№ - заходим на порт
ip source check user-bind enable - включаем правила привязок
undo ip source check user-bind enable - отключаем правила привязок
