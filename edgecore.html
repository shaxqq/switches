sh interfaces status ethernet 1/# - посмотреть состояние порта

sh mac-address-table interface ethernet 1/№ - посмотреть МАС адрес на порту

sh ip dhcp snooping binding – посмотреть кто получает по DHCP

sh system – посмотреть информацию о свиче

sh vlan – посмотреть все Vlan`ы на свиче

show interfaces counters ethernet 1/№ - посмотреть ошибки на порту


Тест кабеля можно сделать не на всех версиях
sh cable-diagnstic interface ethernet 1/№

Добавить Vlan
Заходим на порт
Vty-0(config-if)# switchport mode access
Vty-0(config-if)# switchport allowed vlan add № untagged
Vty-0(config-if)# switchport native vlan № (native - PVID)
Удалить Vlan
Vty-0(config-if)# switchport allowed vlan remove №
